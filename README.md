* [Land Rush](https://conwaylife.com/wiki/OCA:Land_Rush)
* [Mazectric](https://conwaylife.com/wiki/OCA:Mazectric)

## Usage
* **Left/Right** to select rule
* **A** to reshuffle cells
* **Start** to pause
